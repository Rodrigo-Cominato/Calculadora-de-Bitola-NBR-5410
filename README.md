# NBR 5410 Cable Sizing Calculator

This repository contains the source code for an educational cable sizing calculator based on **ABNT NBR 5410**.

## Features

- Cable sizing calculation
- Voltage drop calculation
- Step-by-step calculation details
- PDF export
- Light & Dark mode
- Portuguese & English support

## Technologies

- C++
- WebAssembly
- HTML
- CSS
- JavaScript

> This project was developed exclusively for educational purposes.
